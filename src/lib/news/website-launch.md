## 官网

maincore 官网~~激情上线~~，核心基于 **SvelteKit** + **MDsveX** 构建。

预计会托管在netlify上

### 技术栈

| 层 | 技术 |
|----|------|
| 框架 | SvelteKit |
| 内容 | Markdown + MDsveX |
| 搜索 | MiniSearch (对于中文有手搓的CJK分字) |

### 页面

- **News** — 开发动态
- **Product** — 产品目录
- **Blog** — 技术文章
- **About** — 组织介绍
